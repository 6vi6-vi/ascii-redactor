# ASCII Paint

## Описание проекта

ASCII Paint — это редактор псевдографики для создания ASCII-арта в консоли. Программа позволяет рисовать линии, прямоугольники, выполнять заливку областей, сохранять результаты в файлы и загружать работы из файлов.

## Требования

- CMake 3.10 или выше
- Компилятор с поддержкой C++17
- Windows (используются Windows-специфичные библиотеки conio.h и windows.h)

## Сборка проекта

### Клонирование репозитория
git clone <https://github.com/6vi6-vi/ASCII-redactor.git>

cd ASCII-redactor

### Создание директории для сборки
cmake -S . -B build

### Компиляция
cmake --build build 

### Запуск (Windows)
build\bin\ascii_paint.exe
